The FICL Model

FICL is based on FORTH in that it is a stack machine programmed using RPN. In other words you push 'things' on the stack and use words to operate on them there. For example '1 2 + .' will print 3 - 

  1. Push 1 on the stack
  2. Push 2 on the stack
  3. The + operator pops the two top elements from the stack and pushes the result
  4. Dot prints the top of the stack - the result of the addition.
