;call

 

Guide

( push word -- return )

Call a pre-defined word that has been pushed on the stack. Used when the word to be called needs to be a variable based on other factors.

 

param-1...param-n push word ;call

( instance method -- return )

param-1...param-n instance method ;call \ same as instance.method(param1,...,param-n)

Call a native method (as reference with ;method) in a specific instance of the owning object and a list of pre-defined parmeters.

Feature

;call ( instance method -- return )

Call a predefined method on a given instance. It will only push a value on the stack if the method has one.
 
: debug-on ;ficl ;method debugOn ;ficl ; ;call ; immediate

 

This word retrieves a method from FICL and then immediately calls it. When you are likely to call a native method frequently, it is better to save the method reference:

 

: &" ;method stringLiteral ;ficl ; ; constant
: " ( -- s1 | string literal ) ;ficl &" ;call ; immediate

 

By making the method identifier( &") a constant it is only calculated one, no matter how many times " is used.

Scenario

Call word  (GWT)

Scenario

Call FICL method  (GWT)

This is a convoluted manufactured way to print a number. First we push that number on the stack, then find the print method. At run-time ;call will pull the instance (;ficl), then the method reference and finally the items to print from the stack to produce a result.
