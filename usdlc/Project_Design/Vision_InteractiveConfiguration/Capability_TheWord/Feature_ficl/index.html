;ficl ( -- instance )

Provides access to the FICL engine in the parent environment by placing a reference on the stack. Use it with ;method and ;field to access the FICL public interface.
 
: &print ;ficl ;method print Object ; ; constant

 

Creates a reference to FICL.print(Object message).

Scenario

Check FICL abort status  (GWT)

Abort will always be false from FICL - or the current task would have already failed.
