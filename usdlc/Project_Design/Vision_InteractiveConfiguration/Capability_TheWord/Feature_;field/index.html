;field ;get ;set

 

Feature
;field ( -- field | fieldName className )
;get ( field -- value )
;set (field value -- )
 
Push a reference to an object field from the native language onto the stack. Use ;get and ;set to access it.
Scenario

Get field value  (GWT)

Note that we define &abortField as constant and then reference it with abortField. This is so we don't use reflection more than once to retrieve the field even if called may times.

Scenario

Set field value  (GWT)
