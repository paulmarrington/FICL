constant

Feature
( -- )
 
An immediate word that replaces the last definition with it's result. This allows definitions that don't change to only be evaluated once.
 
: name words ; constant
 
Immediate words constant and variable can be both used for the same word since they have a different meaning.
Scenario

non-constant  (GWT)

Create a word that returns a random number. Run it 3 times and make sure it does not return the same number.

Scenario

constant  (GWT)

Repeat the experiment after making the word constant. This time all the results should be equal because the word was only evaluated once.
