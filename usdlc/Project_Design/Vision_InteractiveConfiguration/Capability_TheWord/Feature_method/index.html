;method

 

Feature
( -- | method class )
 
An immediate word that pulls the names of the method and class from the input stream and uses reflection to create a reference pointer. In run mode the pointer is placed in the stack to be executed with ;call.
 
;method methodName classNameOrReferenceOrInstance param-0 ... param-n ;
 
methodName: as it is used in the native code-base.
classNameOrReferenceOrInstance: A fully qualified class name string or a reference to either the class or an instance of the class.
param-x: The parameter type for code-bases that require it. FICL defines Object, Code, String, Number, Integer and Long.
Guide

;method ( -- | method class )

 

Push a reference to a method of a specified class onto the stack. Use ;call to call the method on an instance of the object.
 
: &. ;method print ;ficl ; ; constant
 
defines a word that pushes a reference to FICL.print() into the stack.
Scenario

The FICL print method  (GWT)

Specifying parameters is useful for calling native methods that do not know of the FICL stack.

Scenario

Boolean conversion  (GWT)

Note the lower case boolean. This provides for unwrapping from Boolean to boolean in the parameter.
