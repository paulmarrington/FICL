;class

Feature
( -- class | class name )
 
Words such as ;method and ;field require a reference to the underlying class. This is one of the few words that are parent language dependent. Put then in a separate file if you need your FICL code to run on more than one language.
 
: FICL ;class usdlc.FICL ;
