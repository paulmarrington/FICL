debug-compile

 

Feature
( -- )
 
The word debug-compile is an immediate word that will write output for each following word as it is compiled. It is useful for discovering problems with immediate words.
 
debug-compile
: newWord ... ;
Scenario

debug  (GWT)

Debug writes compile information to the same output stream as other printing methods, so we can test the results
